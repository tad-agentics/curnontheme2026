## Directory Structure

```

├── core/                   # Core theme functionality
├── partials/               # Reusable template parts/components

├── woocommerce_primary/    # Primary WooCommerce customizations

├── footer.php             # Footer template
```

## Installation

1. Download the theme files
2. Upload to `/wp-content/themes/` directory
3. Activate the theme in WordPress admin panel (Appearance > Themes)
4. Install and activate required plugins
5. Configure theme settings

## Development

### File Structure

- **Template Files:** Root directory contains main WordPress template files
- **Components:** Modular components in `/partials/` directory
- **Assets:** CSS, JS, and images in `/public/` directory
- **Application Logic:** PHP classes and functions in `/app/` and `/core/`

### Customization

- Theme functions: Edit `functions.php`
- Styles: Edit `style.css` or files in `/public/css/`
- Templates: Modify template files or create custom templates in `/page-template/`
- WooCommerce: Override templates in `/woocommerce/` directory
